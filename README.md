# Classic Models Sales and Profit Dashboard | Power BI


## Overview

This project is an interactive Power BI dashboard built using the Classic Models dataset to analyze sales performance, profitability, customer distribution, and order trends. The dashboard enables users to explore business insights through interactive visualizations, filters, and navigation between report pages.

## Tools Used

Power BI | Power Query | DAX | MySQL | Classic Models Dataset

## Dashboard Features

### Page 1 – Sales & Net Profit Analysis

- Interactive toggle button to switch between Sales and Net Profit analysis.
- Date range slicer for filtering data by selected period.
- Product slicer for product-wise analysis.
- KPI cards displaying:
-- Total Sales
-- Count of Unique Orders
-- Average Order Value
- Line charts comparing:
-- Sales between USA and Rest of the World
-- Unique Orders by region
-- Average Order Value by region
- Bar chart showing Sales/Net Profit by Product Line.
- Scatter chart analyzing Sales/Net Profit against Cost of Sales.
- Donut chart displaying Sales/Net Profit by Office.
- Column chart showing Sales/Net Profit by Customer Country.
- Navigation button to move to the detailed analysis page.

### Page 2 – Profit Decomposition & Sales Overview

- Interactive Decomposition Tree for Net Profit analysis with drill-down by:
-- Customer Country
-- Product Line
-- Customer Name
- Sales Overview table including:
-- Year
-- Month
-- Sales Value
-- Month-over-Month (MoM %) Growth
-- Year-to-Date (YTD) Sales

## Key Insights

- Compare business performance between USA and Rest of the World.
- Identify the most profitable product lines and customer regions.
- Analyze relationships between sales, cost, and profit.
- Track monthly sales trends and business growth using MoM% and YTD metrics.
- Explore profit drivers interactively through the decomposition tree.
